## 選定の基準

技術選定では以下を重視：

1. **開発速度** - セットアップと実装の速さ
2. **エコシステム** - ライブラリやツールの充実度
3. **学習コスト** - ドキュメントとコミュニティ
4. **将来性** - 長期的なメンテナンス
